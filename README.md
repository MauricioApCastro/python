# Treino Full-Stack

Repositorio de estudos para revisao de fundamentos de programacao e evolucao para desenvolvimento full-stack.

## Objetivo

Revisar desde o inicio, com exercicios praticos em Python, ate chegar em backend, banco de dados, API e frontend.

## Estrutura

- `aulas/`: codigos criados durante as aulas.
- `anotacoes/`: diario de aulas, conceitos e revisoes.

## Progresso

- Entrada e saida com `input()` e `print()`
- Variaveis e tipos
- Conversao com `float` e `int`
- Condicionais com `if`, `elif`, `else`
- Validacao simples de dados
- Tratamento de erro com `try/except`
- Repeticao com `while`
- Listas e dicionarios
- Introducao a funcoes e `return`
