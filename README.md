# Treino Full-Stack

Repositorio de estudos para revisao de fundamentos de programacao e evolucao para desenvolvimento full-stack.

## Objetivo

Revisar desde o inicio, com exercicios praticos em Python, ate chegar em backend, banco de dados, API e frontend.

## Estrutura

- `aulas/`: codigos criados durante as aulas.
- `anotacoes/`: diario de aulas, conceitos e revisoes.

## Progresso

- Entrada e saida com `input()` e `print()`
- Variaveis e tipos
- Conversao com `float` e `int`
- Condicionais com `if`, `elif`, `else`
- Validacao simples de dados
- Tratamento de erro com `try/except`
- Repeticao com `while`
- Listas e dicionarios
- Introducao a funcoes e `return`
- Fixacao de lista, dicionario, funcao, parametro, `return`, `append` e `for`
- Inicio do menu do sistema com opcoes de cadastro, listagem, total e sair

## Metodo atual

Antes de avancar para conteudo novo, o foco e fixar bem os fundamentos.

O ciclo de estudo combinado e:

1. Apresentar uma necessidade real do sistema.
2. O aluno tenta resolver.
3. O mentor revisa o codigo e corrige a logica.
4. O aluno responde exercicios mentais para confirmar entendimento.
5. So avancar quando o conceito estiver seguro.

Ponto atual: reforco de funcoes, listas e dicionarios usando cadastros de equipamentos.

## Ponto de parada atual

O sistema com menu ja tem estas opcoes funcionando:

- `1` - Cadastrar equipamento
- `2` - Listar todos
